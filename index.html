How it works

01 — Scan

Crawl the live store

Fetches the homepage, product pages, and promos straight from the URL — no plugin, no tag, nothing to install on the site.

02 — Flag

Run two rule engines

Every line of copy is checked against an FTC/FDA claims lexicon; the markup is checked against WCAG 2.2. Each hit is tagged with its regulation, severity, and priority.

03 — Fix

Hand back the rewrite

Each finding shows the flagged text, why it's a risk, and a ready-to-paste Current → Suggested fix. Where a compliant rewrite would cost conversion, it also hands back compliant swaps that still sell — verifiable social proof, survey-backed stats, ingredient specificity — so the copy stays sharp.
